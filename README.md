# API-Operating

### Requisitos e Instalação 
#### Requisitos:
```cdm
Java 11
Apache Maven 3.8.4 
```
#### Instalação 
Idle recomendada Intellij.

No terminal, executar 
```cmd
mvn clean install 
```
Executar o arquivo Main que se encontra no caminho:
```cmd
src/main/java/com/operating/api/Main.java
```
